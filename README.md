# python-project
Understanding Customer Purchasing Behavior Through Diwali Sales Data

🎆 Diwali Sales Analysis
Overview

This project performs Exploratory Data Analysis (EDA) on Diwali sales data using Python. The goal is to understand customer purchasing behavior and identify key factors that influence sales during the festive season.

Tools & Libraries
Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook
Project Workflow
Data Cleaning and Preprocessing
Handling Missing Values
Exploratory Data Analysis (EDA)
Data Visualization
Business Insights Generation
Key Insights
Married women aged 26–35 years are the primary buyers.
Customers from Uttar Pradesh, Maharashtra, and Karnataka contribute significantly to sales.
IT, Healthcare, and Aviation professionals show higher purchasing power.
Food, Clothing, and Electronics are among the most purchased product categories.
